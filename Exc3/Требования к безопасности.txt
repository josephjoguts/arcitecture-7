Требования к безопасности:
    Контроль доступа:
        Доступ только авторизованных пользователей через проверку с tenant-core-app
        Все запросы проходят через SmartHomeIntegrationService
    Шифрование данных:
        Шифрование чувствительных данных в базе данных партнёра
    Мониторинг:
        Ведение логов взаимодействия с внешними сервисами.
    Области видимости API:
         Доступ ограничен исключительно данными по конкретному дому, подъезду или пользователю.


Протоколы аутентификации и авторизации:
    Пользователь -> tenant-core-app: Использует текущий метод авторизации через auth-service-1
    tenant-core-app -> SmartHomeIntegrationService -> Внешний партнер: После авторизации в приложении SmartHomeIntegrationService использует сервисный токен доступа представленный внешним партнером


Организация взаимодействия:
    1) Мобильное приложение авторизируется через auth-service-1
    2) Tenant-core-app по REST передает данные пользователя и регистрацию в желаемом API в SmartHomeIntegrationService
    3) SmartHomeIntegrationService, используя токен партнера для интеграции, отправляет данные пользователя в сервис партнера
